# Parrot-i3-Dotfiles 🇵🇪💻

Mi configuración personal de **i3wm** para Parrot OS, tematizada en **negro y rojo neón** con estilo de ciberseguridad minimalista. 

¡Hecho con cariño para practicar redes y hacking!

## 🚀 Cómo instalar todo en un comando

Si quieres usar este tema en una instalación limpia, abre la terminal y corre esto:

```bash
git clone https://github.com/dzenwertz/Parrot-i3-Dotfiles.git ~/.dotfiles && cd ~/.dotfiles && ./install.sh
```

## ✨ Lo que incluye
- **Mosaicos automáticos**: Las ventanas se acomodan solas en vertical/horizontal según el tamaño de la pantalla (estilo Hyprland).
- **Ventanas flotantes cómodas**: Con `Super + Shift + f` las vuelves flotantes. Puedes arrastrarlas y cambiarles el tamaño con el mouse como en Windows.
- **Tema Perú Cyber**: Todo unificado en negro y rojo neón (Polybar, Kitty y Rofi).
- **Logo patrio en terminal**: Insignia de Perú en arte ASCII al abrir Kitty.
- **Atajos prácticos**: Cambiar fondo de pantalla con `Super + Shift + w`, abrir el gestor de archivos con `Super + e`, y controles de volumen/brillo listos.
